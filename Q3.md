# สถานการณ์และแนวโน้มความต้องการตลาดแรงงานของไทยและต่างประเทศในปัจจุบันและในอนาคตอีก 20 ปีข้างหน้า
## Thailand
### สถานการณ์
    - เศรษฐกิจฟื้นฟูท่องเที่ยวและบริโภคภายในประเทศโอเค , ส่งออกยังมีปัญหา
    - ขาดแรงงานเนื่องจากเข้าสู่สังคมผู้สูงอายุสมบูรณ์
    - แรงงานอิสระมีแนวโน้มเพิ่ม
    - ธุรกิจเล็ก/ออนไลน์ เติบโต
### แนวโน้ม 20 ปีข้างหน้า
    - เติบโตโดยพึ่งพานวัตกรรมเทคโนโลยี และเศรษฐกิจดิจิทัล
    - ตลาดแรงงานเน้น reskill ทุกช่วงวัย
## World
### สถานการณ์
    - อัตราการว่างงานโลก ทรงตัวอยู่ที่ 5%
    - ตลาดแรงงานยืดหยุ่นแต่เศรษฐกิจชะลอตัว -> จำกัดการสร้างงานใหม่
    - เลื่อมล้ำ Africa ว่างงานมากกว่า Europe มาก
### แนวโน้ม 20 ปีข้างหน้า
    - เติบโตโดยพึ่งพานวัตกรรมเทคโนโลยี และเศรษฐกิจดิจิทัล
    - ผู้สูงอายุมีบทบาทมากขึ้นเพราะหลายๆ ประเทศเข้าสู่สังคมสูงอายุ
## ความต้องการแรงงาน
    - 15-24 : เทคโนโลยี และ ภาษา
    - 25-34 : มีข้อมูล ทำด้าน AI , นวัตกรรม
    - 35+ : บริหาร ใช้เชี่ยวชาญเฉพาะทาง ใช้ประสบการณ์ เริ่มทำธุรกิจ
# การเป็นผู้ประกอบการ (Entrepreneurship)
## Thailand
    - SME (Small Medium) : 99.5% ของธุรกิจจดทะเบียน  ทำด้านค้าปรีก/อาหาร/บริการ, GDP ~ 34%
    - ข้อจำกัด : เข้าถึงทุนยาก, ระเบียบซับซ้อน, ขาดทักษะด้าน AI-Digital
## World
    - SME : Tech Start-up, ~55% ของ GDP
    - ข้อจำกัด : แข่งขันสูง มีการเปลี่ยนแปลงเทคโนโลยีเร็วมาก
## แนวโน้มใน 20 ปี
### Thailand 
    - High Digitalized 
    - ท่องเที่ยว, logistic
    - SME -> นวัตกรรม
    - ใช้แรงงานลดลงเพราะ Aging Society
### World
    - Gen AI, IoT, Digital info, Automation
    - Environment Friendly will be trend
    - ลดความเลื่อมล้ำในการเป็นผู้ประกอบการ ทุกคนเป็นได้

# เปรียบเทียบสถานการณ์ระบบการศึกษาและแนวทางการพัฒนาทักษะแรงงานในปัจจุบันของไทยและประเทศอื่นๆ
    - Thailand 
        Normal : ผลิตกำลังคนให้สอดคล้องกับอุตสาหกรรมเป้าหมาย, รณรงค์ Lifelong Learning
        อาชัวศึกษา : เน้นทักษาสมัยใหม่ที่ขาดแคลนเช่น EV, robot มีการทำงานจริงสลับเสมอให้สามารถทำงานได้ทันที
        ครู : ลงมือทำจริงแล้วพัฒนาแทนการอบรบ
    - Japan 
        Normal : Digital Learning , Decentralized Education, Internationalism
        อาชัวศึกษา : เน้นลงมือปฏิบัติ ร่วมมือกับอุตสาหกรรม
        ครู : ฝึกอบรบครูอย่างต่อเนื่อง, มีครูพี่เลี้ยงสำหรับครูใหม่, active learning & student-centred learning
    - Vietnam 
        Normal : ทำให้มีระบบการศึกษาขั้นพื้นฐานครอบคลุมทุกส่วนของประเทศ
        ครู : ได้รับการสนับสนุนจาก WorldBank เปลี่ยนจากอบรบที่ทฤษฎีเป็นลงมือทำ
    - Singapore 
        Normal : คิดวิเคราะห์และแก้ปัญหา แทนการท่องจำ, เรียนรู้หลากหลายภาษาผ่านการจัดการแข่งขันร่วมกับประเทศต่างๆ
        อาชัวศึกษา : มีการฝึกอมรบเต็มเวลา
        ครู : มีการคัดเลือกครู, ปรับเงินเดือนครูใหม่ให้มีแรงทำต่อ
# แนวทางการพัฒนาระบบการศึกษาและการพัฒนาทักษะที่จําเป็นในศตวรรษที่ 21 ของไทยทั้งในระยะสั้น (ไม่เกิน 5 ปี) และในระยะยาว
    - Life-long Learning 
        - สั้น : 
            - ศูนย์พัฒนาและฝึกทักษะตลอดชีวิตในทุกจังหวัด, online ทำให้สามารถเข้าถึงได้
            - Upskill&Reskill สำหรับแรงงานในอุตสาหกรรม
        - ยาว : 
            - Digital Learning Pass เพื่อเก็บข้อมูลทักษะที่น่าเชื่อถือสำหรับสมัครงานและการแนะนำงาน
            - อาชีวศึกษา -> Work-Integrated Learning ที่เชื่อมกับอุตสาหกรรมใหม่ๆ
    - Teacher 
        - สั้น : 
            - อมรม -> พัฒนาครูที่หน้างาน
            - เพิ่มค่าตอบแทนครูในพื้นที่ห่างไกล
        - ยาว : 
            - จัดตั้งสถาบันพัฒนาครูที่เรียนรู้ด้วยเทคโนโลยี ทำให้สามารถเรียนรู้ได้อย่างต่อเนื่อง
            - สร้างเครือข่ายครูให้ครูสามารถพัฒนาหลักสูตรออนไลน์และผลิตสื่อเรียนรู้เชิงสร้างสรรค์ได้
    - อาชีวศึกษา 
        - สั้น : 
            - เพิ่มการเรียนรู้แบบคู่ขนาน เพิ่มประสบการณ์
        - ยาว :
            - ยกระดับอาชีวศึกษาให้เทียบเท่ามหาวิทยาลัยสายอาชีพของสิงคโปห์ เพื่อสร้างแรงงานฝีมือคุณภาพ
    - ระยะสั้น 
        - AI พัฒนาเร่งด่วนเพื่อรองรับตลาดแรงงานปัจจุบัน
            - หลักสูตรระยะสั้น 1-6 เดือน เรียนออนไลน์ + ปฎิบัติจริง
        - อบรบอาชีพระยะสั้น 
            - 1-3 เดือน เช่น อาหาร เครื่องดืม งานช่าง งานบริการ 
            - เปิดศูนย์ฝึกอาชีพในทุกจังหวัด
    - ระยะยาว 
        - ธนาคารหน่วยกิตแห่งชาติ -> เรียนรู้ได้ตลอดชีวิต
            - Course : มหาวิทยาลัย ที่รัฐยอมรับ และสามารถรับรองให่้ได้
            - เรียนได้หน่วยกิต -> หน่วยกิตแลกวุฒิ
