# ข้อมูลทางด้านประชากร ข้อมูลด้านเศรษฐกิจและสังคม และข้อมูลด้านการสาธารณสุข ในเมียนมาร์ กัมพูชา และลาว
## เมียนมาร์
### เศรษฐกิจ
    - เกษตร(25% GDP, 70% Pop) และส่งออกทรัพยากรธรรมชาติ
### สังคม
    - ขัดแย้งภายในประเทศ
    - งบประมาณด้านการศึกษาน้อย
    - อัตราการรู้หนังสือต่ำ
### สาธารณสุข
    - สุขภาพประชากรต่ำกว่ามาตรฐานภูมิภาค
    - ขาดแคลนทรัพยากรและบุคลากรทางการแพทย์
    - โรคต่างๆ เพิ่มขึ้นอย่างต่อเนื่อง
    - ความเหลื่อมล้ำสูงในเรื่องบริการสุขภาพ

## กัมพูชา
### เศรษฐกิจ
    - การท่องเที่ยว (39% GDP), ส่งออกสิ่งทอ, เกษตร (20% GDP 33% Pop)
### สังคม
    - แรงงานผิดกฎหมายตามแนวชายแดน
### สาธารณสุข
    - พึ่งพาการรักษาในต่างประเทศ
    - ความเหลื่อมล้ำสูงในเรื่องบริการสุขภาพ(น้ำสะอาด/ห้องน้ำสะอาด)
    - งบประมาณไม่พอ/บริหารไม่ดี

## ลาว
### เศรษฐกิจ
    - อุตสาหกรรมและพลังงาน (33% GDP), เกษตร (20% GDP 65-70% Pop)
### สังคม
    - หนี้สาธารณะที่เพิ่มขึ้นจากการลงทุนโครงสร้างพื้นฐานขนาดใหญ่ รถไฟลาว-จีน
### สาธารณสุข
    - โรคไม่ติดต่อสูง
    - งบประมาณด้านสุขภาพไม่พอ

## โดยรวม
    - ประชากร : วัยแรงงาน ~ 65% 
    - เศรษฐกิจ : รายได้เฉลี่ยต่ำ มีความเหลื่อมล้ำระหว่างชนบทกับเมืองอยู่มาก 
    - สังคม : อัตราการรู้หนังสือต่ำ งบประมาณสำหรับการศึกษามีจำกัด
    - สาธารณสุข : งบประมาณจำกัด ขาดแคลนทรัพยากรและบุคลากรทางการแพทย์ โรงบาลเอกชนกระจุกอยู่ตามเมืองใหญ่ 

# นโยบายไทยปัจจุบัน
    - ระบบ MOU : ประเทศ-ประเทศ ให้แรงงานเข้ามาทำงานได้อย่างเป็นระบบ โปร่งใส ตรวจสอบได้
    - มาตรการผ่อนผันแรงงานผิดกฎหมาย : ให้ออกในเวลาที่กำหนด, ให้โอกาสลงทะเบียน
    - One Stop Service OSS : รวมกระบวนต่างๆ ที่จำเป็นไว้ในจุดบริการเดียว อำนวยความสะดวก ลดขั้นตอน เพิ่มประสิทธิภาพ 
    - ลดค่าธรรมเนียมการลงตรา (Visa), เพิ่มความสะดวกในการขอใบอนุญาตทำงาน e-work permit
    - ยังไม่มีนโยบายชัดเจนในการให้ PR หรือแปลงสัญชาติ เน้นการบริหารจัดการแบบชั่วคราวและหมุนเวียน

# ผลกระทบเชิงบวก และผลกระทบเชิงลบของแรงงานต่างด้าว
## เชิงบวก 
    - เติมเต็มช่องว่างงานแรงงาน
    - ส่งเสริม GDP
    - ลดต้นทุนการผลิต
## เชิงลบ 
    - กดดันค่าจ้างงานในตลาดแรงงาน เกิดการแข่งขันในตลาดแรงงานระดับล่าง (แรงงานต่างด้าวเพิ่ม 10% ค่าจ้างเฉลี่ยลดลง 0.23%)
    - ภาระต่อระบบสาธารณสุขของไทย
    - อาจเป็นแหล่งโรคติดเชื้อแพร่เพราะไม่มีการตรวจสุขภาพ
    - ปัญหาสังคมและความมั่นคง
# ควรสนับสนุนไหม 
    - ควร เพราะมีบทบาทสำคัญต่อการขับเคลื่อน GDP อย่างยิ่งในเกษตร/ก่อสร้าง/ประมง ที่เป็นแรงงานไทยมีแนวโน้มลดลงเรื่อยๆ นอกจากนี้ ไทยยังเข้าสู่ภาวะสังคมผู้สูงอายุอย่างสมบูรณ์ แต่ต้องอยู่ใต้ระบบการบริหารจัดการอย่างเป็นระบบ โปร่งใส
# ปัญหาที่ต่างด้าวเจอ 
    - ขึ้นทะเบียนซับซ้อน
    - ละเมิดสิทธิมนุษยชน เข้าไม่ถึงสิทธิขั้นพื้นฐาน
    - ขาดหลักประกันทางสุขภาพ
# แผนการปรับเปลี่ยนนโยบาย 
    - ปรับระบบการบริหารจัดการให้เป็นระบบและยืดหยุ่น
    - จัดตั้งฐานข้อมูลให้สามารถตรวจสอบสถานะได้  realtime -> ชักจูงให้พวกเขาเข้าสู่ระบบที่ถูกต้องด้วย
    - ขยายสิทธิให้แรงงานทุกคน
    - อบรมแรงงานของก่อนเข้างาน ยกระดับแรงงาน
